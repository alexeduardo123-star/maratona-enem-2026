# Conteúdos — História

Coloque aqui arquivos `.txt`, `.md` ou `.pdf` com os conteúdos desta matéria para serem importados pelo app.

## Sugestões de tópicos

- Brasil Colonial (capitanias, Ciclo do Ouro)
- Brasil Império (independência, aboliconismo)
- República Velha (Política do Café com Leite)
- Era Vargas e Estado Novo
- Ditadura Militar (1964-1985)
- História Geral (Revolução Francesa, Guerras Mundiais, Guerra Fria)

## Como usar

Adicione um arquivo por tópico com o nome do tema. Ex: `era-vargas.md`

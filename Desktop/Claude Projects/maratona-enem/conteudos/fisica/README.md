# Conteúdos — Física

Coloque aqui arquivos `.txt`, `.md` ou `.pdf` com os conteúdos desta matéria para serem importados pelo app.

## Sugestões de tópicos

- Cinemática (MRU, MRUV, queda livre)
- Dinâmica (Leis de Newton)
- Termodinâmica (leis, ciclos)
- Eletrostática e Eletrodinâmica
- Ondulatória e Ótica
- Física Moderna (relatividade, quântica)

## Como usar

Adicione um arquivo por tópico com o nome do tema. Ex: `leis-de-newton.md`

# Conteúdos — Filosofia

Coloque aqui arquivos `.txt`, `.md` ou `.pdf` com os conteúdos desta matéria para serem importados pelo app.

## Sugestões de tópicos

- Filosofia Antiga (Sócrates, Platão, Aristóteles)
- Filosofia Medieval e Moderna
- Ética e Moral (Kant, Aristóteles, utilitarismo)
- Filosofia Política (contratualismo, Marx)
- Epistemologia e Teoria do Conhecimento
- Existencialismo e Filosofia Contemporânea

## Como usar

Adicione um arquivo por tópico com o nome do tema. Ex: `etica-kantiana.md`

# Conteúdos — Artes

Coloque aqui arquivos `.txt`, `.md` ou `.pdf` com os conteúdos desta matéria para serem importados pelo app.

## Sugestões de tópicos

- Arte Barroca Brasileira (Aleijadinho, Mestre Ataíde)
- Modernismo Brasileiro (Semana de 22, Tarsila do Amaral)
- Arte Contemporânea
- Cinema e Audiovisual (Cinema Novo, Cinema Marginal)
- Música Popular Brasileira (Bossa Nova, Tropicalismo, MPB)
- Artes Visuais Mundiais

## Como usar

Adicione um arquivo por tópico com o nome do tema. Ex: `tropicalismo.md`

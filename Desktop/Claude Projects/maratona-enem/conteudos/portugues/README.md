# Conteúdos — Língua Portuguesa

Coloque aqui arquivos `.txt`, `.md` ou `.pdf` com os conteúdos desta matéria para serem importados pelo app.

## Sugestões de tópicos

- Interpretação de Texto
- Gramática (concordância, regência, crase)
- Semântica e Estilística
- Figuras de Linguagem
- Gêneros Textuais (crônica, editorial, conto, artigo)

## Como usar

Adicione um arquivo por tópico com o nome do tema. Ex: `interpretacao-de-texto.md`

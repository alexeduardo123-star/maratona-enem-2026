# Conteúdos — Literatura

Coloque aqui arquivos `.txt`, `.md` ou `.pdf` com os conteúdos desta matéria para serem importados pelo app.

## Sugestões de tópicos

- Barroco e Arcadismo
- Romantismo (Alencar, Álvares de Azevedo)
- Realismo e Naturalismo (Machado de Assis, Aluísio Azevedo)
- Pré-Modernismo (Euclides da Cunha, Lima Barreto)
- Modernismo (Semana de 22, Drummond, Guimarães Rosa)
- Contemporaneidade

## Como usar

Adicione um arquivo por tópico com o nome do tema. Ex: `machado-de-assis.md`

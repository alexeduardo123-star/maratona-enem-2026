# Conteúdos — Química

Coloque aqui arquivos `.txt`, `.md` ou `.pdf` com os conteúdos desta matéria para serem importados pelo app.

## Sugestões de tópicos

- Ligações Químicas (iônica, covalente, metálica)
- Estequiometria e Cálculos
- Química Orgânica (funções, nomenclatura, reações)
- Soluções e Concentrações
- Equilíbrio Químico
- Eletroquímica

## Como usar

Adicione um arquivo por tópico com o nome do tema. Ex: `quimica-organica-funcoes.md`

# Conteúdos — Matemática

Coloque aqui arquivos `.txt`, `.md` ou `.pdf` com os conteúdos desta matéria para serem importados pelo app.

## Sugestões de tópicos

- Funções (linear, quadrática, exponencial, logarítmica)
- Geometria Plana e Espacial
- Probabilidade e Estatística
- Álgebra e Equações
- Trigonometria
- Progressões Aritméticas e Geométricas

## Como usar

Adicione um arquivo por tópico com o nome do tema. Ex: `funcoes-quadraticas.md`
